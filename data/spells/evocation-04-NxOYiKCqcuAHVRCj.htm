Name: Transcribe Moment Nom: Instant transcrit État: changé État d'origine: libre ------ Description (en) ------ NE PAS TRADUIRE ------ Description (fr) ------

Vous créez instantanément un parchemin petit et permanent qui contient une description magique précise de ce qui vous entoure dans chaque direction. N'importe quelle créature qui passe 1 minute à lire le parchemin - inscrit dans une langue de votre choix que vous connaissez - voit immédiatement et fait l'expérience de votre environnement au moment où vous avez enregistré (incluant tous les évènements qui se sont produits durant ce round) exactement comme vous les avez vécu.

Cette description enregistre seulement ce que vous pouvez ressentir vous-même. Par exemple, si vous ne pouvez pas voir les créatures invisibles, l'enregistrement ne les révèlera pas même à ceux qui pourraient les voir habituellement ; un lecteur ne peut utiliser sa Perception pour remarquer des détails que vous avez manqué, mais peuvent essayer un test pour Se souvenir basé sur ce dont ils ont l'expérience s'ils ont une connaissance dont vous manquez.

Vous pouvez avoir plusieurs parchemins d'Instant transcrit au même moment mais avec un maximum égal à votre modificateur d'Intelligence. Si vous Lancez le sort encore alors que vous avez atteint le nombre maximum de parchemins, choisissez un des actuels parchemins qui perd sa magie puis créez le nouveau.
