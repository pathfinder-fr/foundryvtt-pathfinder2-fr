Name: Terrain Transposition Nom: Transposition de terrain État: changé État d'origine: libre ------ Data ------ RangeEN: 90 feet RangeFR: 27 m ------ Description (en) ------ NE PAS TRADUIRE ------ Description (fr) ------

Vous transposez mystiquement votre lieu actuel avec un autre. Lorsque vous êtes dans un environnement naturel, vous vous transportez vous-même et les objets que vous portez vers une case libre que vous pouvez voir à portée. Si vous avez @Compendium[pf2e.feats-srd.sgo7J9BVofBqwlsF]{Environnement de prédilection} et que vos deux destinations de départ et d'arrivée sont sur votre terrain de prédilection, la portée passe à 30 mètres. Si vous possédez un compagnon animal et qu'il vous est adjacent, vous pouvez le transporter avec vous sur une case adjacente libre, mais si ce sort devrait emmener toute autre créature avec vous, même si elle se trouve dans un contenant extradimensionnel, le sort est perdu.
