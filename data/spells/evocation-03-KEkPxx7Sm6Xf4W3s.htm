Name: Blazing Dive Nom: Plongeon ardent État: changé État d'origine: libre ------ Description (en) ------ NE PAS TRADUIRE ------ Description (fr) ------

L'air surchauffé s'amasse en dessous de vous, vous permettant de flotter haut dans le ciel avant de retomber comme une comète enflammée. Vous volez de 4,5 m dans les airs puis Volez en ligne droitejusqu'à une case vide de votre choix dans les 18 m. Lorsque vous atterrissez, l'air accumulé explose à partir de vous, infligeant 3d4 dégâts contondants et 3d6 dégâts de feu (jet de Réflexes basique) à toutes les créatures dans une émanation de 3 m.

Intensifié (+1) Les dégâts contondants augmentent de 1d4 et les dégâts de feu de 1d6.
