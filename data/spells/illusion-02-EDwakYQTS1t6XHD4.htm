Name: Umbral Extraction Nom: Extraction ombrale État: changé État d'origine: libre ------ Description (en) ------ NE PAS TRADUIRE ------ Description (fr) ------

Vous vous préparez à voler les sorts de vos ennemis, l'arrachant à partir de l'ombre de votre ennemi pour la placer dans une poche éphémère sur le plan de l'Ombre. Vous obtenez un emplacement de sort temporaire, que vous ne pouvez remplir qu'en volant un sort à un incatateur ennemi. Pendant la durée d'Extraction ombrale, vous pouvez utiliser l'action Voler pour essayer de prendre un des sorts préparés ou un sort spontané inutilisé au lieu d'un objet. Vous pouvez aussi faire une tentative de Voler au moment où vous Lancez Extraction ombrale. Si vous obtenez un succès à votre test pour Voler un sort, vous infligez [[/r 1d4 #mentaux]]{1d4 dégâts mentaux} à la cible par niveau du sort volé lié à la taxe psychologique de vide spirituel créé en arrachant le sort et vous placez le sort volé dans votre emplacement de sort temporaire. Il obtient le trait ombre s'il est lancé à partir de cet emplacement.

Déterminez le sort volé de manière aléatoire parmi les sorts qui ont un niveau de moins qu'Extraction ombrale ou dans les sorts qui ont le même niveau si vous avez obtenu un succès critique. Si la cible n'a pas d'emplacement de sort de ce niveau, utilisez le niveau le plus haut dans lequel la cible possède un emplacement de sort. (Si la cible n'a pas de sort éligible, vous n'obtenez rien, mais vous pouvez continuer à essayer de Voler des sorts pendant la durée restante). Si la cible est un incantateur préparé, vous volez de manière aléatoire un de ses sorts préparés. Si la cible est un incantateur spontané, vous volez à la place un emplacement de sort non dépensé du niveau approprié et déterminez le sort de manière aléatoire à partir des sorts connus de cette cible. Vous ne pouvez ni voler un sort inné de la cible, ni un sort de focalisation, ni un tour de magie. Vous savez quel est le sort dès que vous l'avez volé et votre emplacement de sort temporaire devient un emplacement de sort du même niveau que le sort que vous avez volé.

Comme à l'ordinaire, vous ne pouvez seulement Voler qu'une cible à votre portée. Une fois que vous avez placé un sort dans votre emplacement de sort temporaire, vous ne pouvez tenter de Voler plus de sorts. Si vous avez obtenu un succès à votre test de Vol, vous devez prendre le sort que vous avez volé. Même si vous êtes un incantateur spontané, vous ne pouvez utiliser l'emplacement de sort temporaire que pour lancer le sort volé et vous pouvez lancer seulement le sort volé à partir de votre emplacement de sort temporaire. Lorsqu'Extraction ombrale se termine, vous perdez l'emplacement de sort temporaire et tout sort toujours emmagasiné à l'intérieur. Lorsque vous Lancez Extraction ombrale, toute incantation précédente d'Incantation ombrale qui vous affecte se termine.
