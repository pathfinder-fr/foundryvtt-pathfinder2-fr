Name: Time Jump Nom: Saut temporel État: changé État d'origine: libre ------ Description (en) ------ NE PAS TRADUIRE ------ Description (fr) ------

Vous faites un bond en avant de quelques secondes dans le temps, apparaissant sur le champ de bataille en un clin d'œil. Vous gagnez 2 actions, chacune d'entre elles devant être utilisée pour Bondir, Se relever, Faire un pas ou Marcher rapidement. Si vous avez une Vitesse appropriée, vous pouvez ajouter @Compendium[pf2e.actionspf2e.H6v1VgowHaKHnVlG]{Creuser}, @Compendium[pf2e.actionspf2e.pprgrYQ1QnIDGZiy]{Escalader}, @Compendium[pf2e.spells-srd.A2JfEKe6BZcTG1S8]{Voler}, ou @Compendium[pf2e.actionspf2e.c8TGiZ48ygoSPofx]{Nager} à cette liste. Pendant que vous effectuez ces actions, le temps s'arrête. Toutes les autres créatures sont complètement inconscientes de vos actions, ne peuvent pas parler et ne peuvent pas utiliser d'actions qui seraient déclenchées par votre déplacement.

Pendant que vous effectuez ces actions, vous ne pouvez effectuer aucune autre action, y compris celles qui seraient déclenchées par les actions de déplacement. Une fois les actions terminées, le temps recommence, et pour les spectateurs, il semble que vous vous soyez soudainement téléporté sur la distance que vous avez parcourue. Bondir dans le temps est désorientant, donc si vous utilisez Saut temporel à nouveau dans la minute qui suit l'utilisation du sort, vous devenez @Compendium[pf2e.conditionitems.e1XGnhKNSQIm5IXg]{Stupéfié 4} pendant 1 minute.
