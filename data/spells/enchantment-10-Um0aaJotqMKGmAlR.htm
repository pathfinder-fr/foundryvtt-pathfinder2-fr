Votre représentation entraîne ceux qui l'entendent, les obligeant à vous suivre éperdus d'amiration. Chaque créature dans l'émanation doit effectuer un jet de Volonté lorsque vous Lancez le Sort ou la première fois qu'elle pénètre dans la zone, après quoi elle devient temporairement immunisée pendant 1 journée.

Une fois par tour, vous pouvez dépenser une action unique, qui possède le trait concentration, pour augmenter le rayon de l'émanation de 1,5 m. Vous pouvez @Compendium[pf2e.actionspf2e.AjLSHZSWQ90exdLo]{Révoquer le sort}.

Succès critique La créature n'est pas affectée.

Succès La créature est @Compendium[pf2e.conditionitems.AdPVz7rbaVSRxHFg]{Fascinée} par vous.

Échec La créature dépense toutes ses actions pour se déplacer vers vous pour vous complimenter sur votre représentation. Cet effet s'interrompt si une action hostile est utilisée contre la créature affectée.

Échec critique La cible obtient le trait sbire est elle est @Compendium[pf2e.conditionitems.9qGBRpbX9NEwtAAr]{Contrôlée} par vous. Cet effet s'achève si une action hostile est utilisée contre la créature affectée ou si vous ordonnez à la créature d'utiliser une action qui lui causerait du tort.
