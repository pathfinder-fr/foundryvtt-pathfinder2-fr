Name: Instant Armor Nom: Armure instantanée État: changé État d'origine: libre ------ Description (en) ------ NE PAS TRADUIRE ------ Description (fr) ------

Lorsque vous lancez ce sort, l'armure que vous portez est emportée dans un espace extradimensionnel qui vous est lié. Si l'armure est magique et investie par vous, elle reste investie pendant qu'elle se trouve dans cet espace, bien que vous ne bénéficiez pas de ses avantages. Vous gagnez alors l'action @Compendium[pf2e.actionspf2e.8b63qGzqSUENQgOT]{Retour d'armure} ; une fois l'action utilisée, le sort prend fin. Si l'action n'a pas été utilisée au moment où la durée du sort prend fin, l'espace extradimensionnel s'effondre, rejetant les pièces de l'armure sur le sol sous vos pieds.
