Name: Continual Flame Nom: Flamme éternelle État: changé État d'origine: officielle ------ Description (en) ------

A magical flame springs up from the object, as bright as a torch. It doesn't need oxygen, react to water, or generate heat.

Heightened (+1) The cost increases as follows:

  • 16 gp for 3rd level;
  • 30 gp for 4th,
  • 60 gp for 5th,
  • 120 gp for 6th,
  • 270 gp for 7th,
  • 540 gp for 8th,
  • 1,350 gp for 9th, and
  • 3,350 gp for 10th.
------ Description (fr) ------

Une flamme magique jaillit de l’objet, aussi vive qu’une torche. Elle n’a pas besoin d’oxygène, ne réagit pas en présence d’eau et ne génère aucune chaleur.

Intensifié (+1) Le coût augmente comme suit : 16 po pour le 3e niveau ; 30 po pour le 4e, 60 po pour le 5e, 120 po pour le 6e, 270 po pour le 7e, 540 po pour le 8e, 1 350 po pour le 9e et 3 350 po pour le 10e.
