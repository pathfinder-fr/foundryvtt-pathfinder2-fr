Zone une case de 1,5 m

Vous liez temporairement un sort hostile dans un symbole. Pendant que vous lancez ce sort, vous lancez également un sort d'un niveau inférieur pour le stocker dans le glyphe. Le sort stocké doit prendre 2 actions ou moins pour être lancé, avoir un effet hostile et cibler une créature ou avoir une zone. Toute créature qui entre dans la zone de Glyphe temporaire active le glyphe, libérant le sort nuisible qu'il contient. Vous pouvez définir un mot de passe pour le glyphe. Le prononcer en entrant dans la zone du sort empêche le glyphe de se déclencher. Vous pouvez également définir un déclencheur plus spécifique pour limiter les types de créatures qui déclenchent le glyphe.

Une fois qu'un sort est stocké dans le glyphe, celui-ci acquiert tous les traits de ce sort. Si le sort stocké peut cibler une ou plusieurs créatures, il cible la créature qui a déclenché le glyphe. S'il possède une zone, cette zone est centrée sur la créature qui a déclenché le glyphe. Glyphe temporaire prend fin lorsque le glyphe est déclenché. Le glyphe est un piège magique, utilisant votre DD de sort à la fois pour le test de Perception pour le remarquer et le test de Vol pour le désamorcer ; ces deux tests nécessitent que la créature qui les tente soit qualifiée pour les réussir.

Vous pouvez révoquer Glyphe temporaire.
