Name: Perfected Mind Nom: Esprit perfectionné État: changé État d'origine: officielle ------ Description (en) ------ NE PAS TRADUIRE ------ Description (fr) ------

Vous méditez sur la perfection pour effacer toutes les distractions de votre esprit. Effectuez un nouveau jet de Volonté contre un effet mental vous affectant actuellement qui nécessite un jet de Volonté. Utilisez le résultat de ce nouveau jet de sauvegarde pour déterminer l’issue de l’effet mental, sauf si le nouveau jet de sauvegarde a un résultat pire que l’original, auquel cas rien ne se produit. Vous pouvez utiliser esprit perfectionné contre un effet donné une seule fois.
