Name: Read the Air Nom: Lire l'ambiance État: changé État d'origine: libre ------ Description (en) ------ NE PAS TRADUIRE ------ Description (fr) ------

Vous respirez profondément en examinant une situation sociale, faisant preuve de courtoisie envers tous ceux qui vous entourent tandis que votre intuition capte rapidement des indices sur le contexte social et les comportements implicites. Votre langage corporel change inconsciemment pour tirer parti de ces informations et les utiliser dans vos propres interactions avec ces créatures.

Lorsque vous lancez ce sort, vous faites un test pour Vous souvenir en utilisant Société pour obtenir des informations sur la situation sociale immédiate. Vous gagnez également un bonus de statut de +1 à votre prochain test de Diplomatie pour Faire bonne impression sur les créatures présentes lorsque vous lancez ce sort, à condition que le test ait lieu pendant la durée du sort. Vous ne pouvez utiliser Lire l'ambiance qu'une seule fois dans une situation sociale donnée ; relancer le sort n'a aucun effet.
