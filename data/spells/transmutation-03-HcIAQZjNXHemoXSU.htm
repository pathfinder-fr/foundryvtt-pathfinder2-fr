Vous provoquez la surface à se soulever. La zone devient un terrain difficile et toute trace dans la zone est détruite par le sol agité. Les créatures qui se tiennent dans la zone subissent une pénalité de statut de -1 pour les tests d'Acrobaties pour @Compendium[pf2e.actionspf2e.M76ycLAqHoAgbcej]{Garder l'équilibre} et les @Compendium[pf2e.actionspf2e.21WIfSu7Xd7uKqV8]{Déplacements acrobatiques} et les tests d'Athlétisme pour le @Compendium[pf2e.actionspf2e.2HJ4yuEFY1Cast4h]{Saut en hauteur} et le @Compendium[pf2e.actionspf2e.JUvAvruz7yRQXfz2]{Saut en longueur}. Lorsque vous Maintenez le sort, vous pouvez déplacer la zone agitée jusqu'à 3 mètres dans toute direction. Les créatures immobilisées par le sort sont transportées le long avec les sables mouvants dans la même direction, si possible ; ce déplacement est un déplacement forcé. Les créatures qui entrent ou commencent leur tour en se tenant sur le sable mouvant doivnt tenter un jet de Réflexes.
