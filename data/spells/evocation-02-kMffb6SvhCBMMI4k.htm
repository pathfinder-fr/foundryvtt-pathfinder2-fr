Sélectionnez un trait élémentaire : air, eau, feu ou terre. La zone élémentaire obtient le trait que vous avez choisi. Vous imprégnez la zone de l'énergie brute de cet élément, créant une zone qui émet et amplifie la magie de ce type. La température peut soudainement augmenter ou diminuer, un nuage d'orage peut se former, etc. Les sorts avec le trait élémentaire choisi lancés contre des créatures dans la zone bénéficient d'un bonus de statut de +2 à un type de dégâts que le sort inflige en fonction du trait choisi : contondants ou électricité pour l'air, contondants ou froid pour l'eau, feu pour le feu et contondants pour la terre. Le lanceur de sorts choisit le type auquel il ajoute le bonus si le sort inflige plus d'un type éligible. Ce bonus est réduit de moitié si le sort n'a pas utilisé d'emplacement de sort (comme un tour de magie, un sort focalisé ou un sort inné).

La zone n'a aucun effet sur le sort si celui-ci n'inflige pas de dégâts d'un type éligible.

Intensifié (+2) Le bonus de statut augmente de 1.
