Name: Secret Page Nom: Page secrète État: changé État d'origine: officielle ------ Data ------ RangeEN: touch RangeFR: contact TargetEN: 1 page up to 3 square feet in size TargetFR: 1 page d'environ 1 mètre carré maximum ------ Description (en) ------ NE PAS TRADUIRE ------ Description (fr) ------

Vous modifiez le texte de la cible en un texte entièrement différent. Si le texte est un grimoire ou un parchemin, vous pouvez le changer pour afficher un sort que vous connaissez du niveau de Page secrète ou inférieur. Le sort de remplacement ne peut être ni lancé ni utilisé pour préparer un sort. Vous pouvez aussi transformer le texte en un texte que vous avez écrit ou auquel vous avez accès. Vous pouvez spécifier un mot de passe permettant à une créature touchant la page de changer le texte dans un sens ou dans l’autre. Vous devez choisir le texte de remplacement et le mot de passe, le cas échéant, quand vous Lancez le sort.
