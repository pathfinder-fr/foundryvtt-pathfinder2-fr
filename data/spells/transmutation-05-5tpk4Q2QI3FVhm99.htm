Name: Rewinding Step Nom: Retour en arrière État: changé État d'origine: libre ------ Description (en) ------ NE PAS TRADUIRE ------ Description (fr) ------

Vous ancrez votre position dans le temps de manière à pouvoir rapidement revenir sur vos pas plus tard avec une complexe chronomancie. La case dans laquelle vous vous trouvez lorsque vous lancez le sort devient votre ancre temporelle. Tant que vous êtes à 150 m de votre ancre temporelle, vous pouvez dépenser une unique action qui a le trait concentration pour revenir dans cette case instantanément (Vous n'avez pas besoin d'avoir cette case en ligne de mire). Lee sort s'achève alors. Vous ramenez n'importe quel objet en votre possession lorsque vous entreprenez le retour.

Si la case qui sert d'ancre est occupée lorsque vous tentez d'y revenir ou si ce sort devrait emporter une autre créature avec vous, le sort échoue à vous transporter, puis se termine.

Intensifié (7e) Vous pouvez revenir si vous êtes dans les 300 m de la case qui vous sert d'ancre.

Heightened (9e) Vous pouvez revenir si vous êtes dans les 1,5 km de la case qui vous sert d'ancre.
