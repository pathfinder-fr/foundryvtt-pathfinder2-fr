Parent Name: Giant Octopus Name: Writhing Arms Nom: Membres grouillants État: officielle ------ Description (en) ------

The giant octopus makes up to four Strikes with different arms, each against a different target. Each attack counts separately for the octopus's multiple attack penalty, but the penalty doesn't increase the penalty until the octopus has made all the attacks. If the octopus subsequently uses the @Compendium[pf2e.bestiary-ability-glossary-srd.Tkd8sH4pwFIPzqTr]{Grab} action, it can Grab any number of creatures it hit with Writhing Arms.

------ Description (fr) ------

La pieuvre géante effectue jusqu’à quatre Frappes avec différents bras, chaque attaque se faisant contre une cible différente. Chaque attaque compte dans le malus d’attaques multiples de la pieuvre géante mais ce malus n’augmente qu’une fois toutes les attaques effectuées. Si la pieuvre enchaîne en utilisant l’action @Compendium[pf2e.bestiary-ability-glossary-srd.Tkd8sH4pwFIPzqTr]{Agripper/Empoigner}, elle Agrippe le nombre de créatures de son choix parmi celles qui ont été touchées par Membres grouillants.
