Parent Name: Baomal Name: Two-Headed Strike Nom: Frappe à deux têtes État: officielle ------ Description (en) ------

The baomal makes a Strike with each set of jaws, each against a different creature. These Strikes count as one attack for the baomal's multiple attack penalty, and the penalty doesn't increase until after both attacks.

------ Description (fr) ------

Le baomal effectue une Frappe avec chaque mâchoire contre deux créatures différentes. Ces Frappes comptent comme une seule attaque pour le malus d’attaques multiples du baomal et le malus n’augmente pas avant que les deux attaques aient été effectuées.
