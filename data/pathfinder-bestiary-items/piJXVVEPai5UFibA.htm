Parent Name: Drakauthix Name: Sporesight (Imprecise) 60 feet Nom: Vision par les spores (imprécis) 18 m État: officielle ------ Description (en) ------

The drakauthix exudes a cloud of spores that it uses to see. This is an imprecise sense that functions only in areas without strong wind currents. Sporesight does not function underwater.

------ Description (fr) ------

Le drakauthix libère un nuage de spores dont il se sert pour voir. C’est un sens imprécis qui ne fonctionne que dans des zones où il n’y a pas un fort courant d’air. Ce pouvoir ne fonctionne pas sous l’eau.
