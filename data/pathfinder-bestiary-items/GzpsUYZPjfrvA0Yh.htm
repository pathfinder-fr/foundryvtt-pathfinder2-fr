Parent Name: Poracha Name: Object Meld Nom: Fusion avec les objets État: officielle ------ Description (en) ------

The poracha touches an adjacent object of enough volume to contain it. It merges into the object indefinitely, and while it's inside, time moves exceptionally slowly for it and it has only limited perception of the outside world. The poracha can sense what's going on outside the object only using tremorsense. The poracha can use only mental actions or Jaunt; if it Jaunts, it leaves the object and appears where it chooses. If a creature Strikes the object, the poracha is expelled unharmed.

------ Description (fr) ------

Le poracha touche un objet adjacent qui a un volume suffisant pour le contenir. Il fusionne avec cet objet indéfiniment et tant qu’il est à l’intérieur, le temps passe pour lui exceptionnellement lentement et il n’a qu’une perception limitée du monde extérieur. Le poracha ne peut utiliser que perception des vibrations pour savoir ce qui se passe à l’extérieur de l’objet. Il ne peut utiliser que des actions mentales ou Escapade ; s’il utilise Escapade, il quitte l’objet et réapparaît où il veut. Si une créature effectue une Frappe contre l’objet, le poracha en est expulsé indemne.
