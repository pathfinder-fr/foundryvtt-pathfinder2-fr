Parent Name: Dezullon Name: Root Nom: Enracinement État: officielle ------ Description (en) ------

Until the next time it acts, the dezullon appears to be a normal pitcher plant. It has an automatic result of 41 (44 in forests or swamps) on Deception checks and DCs to pass as a non-creature plant.

------ Description (fr) ------

Jusqu’à sa prochaine action, la dézullone a l’air d’une plante carnivore normale. C’est un résultat automatique de 41 (44 dans les forêts et les marais) pour les tests de Duperie et les DD pour se faire passer pour une plante normale.
