Parent Name: Quelaunt Name: Rapid Strikes Nom: Frappes rapides État: officielle ------ Description (en) ------

The quelaunt makes three melee Strikes, each against a different target within reach. The multiple attack penalty applies to each attack, but increases only after all the attacks have been made.

------ Description (fr) ------

Le quelaunt porte trois Frappe au corps-à-corps. Chacune est dirigée contre une cible différente qui se trouve à portée. La pénalité d’attaques multiples s’applique à chaque attaque mais n’augmente qu’une fois que toutes les attaques ont été portées.
