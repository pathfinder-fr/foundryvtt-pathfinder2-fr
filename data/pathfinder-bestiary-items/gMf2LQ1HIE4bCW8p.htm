Parent Name: Web Lurker Name: Spring upon Prey Nom: Bondir sur sa proie État: officielle ------ Description (en) ------

Trigger A creature touches the web lurker's web while the web lurker is on it.

Requirement Initiative has not yet been rolled.

Effect The web lurker automatically notices the creature and Strides or Climbs before rolling initiative.

------ Description (fr) ------

Conditions L’initiative n’a pas encore été déterminée.

Déclencheur Une créature touche la toile du rôdeur des toiles pendant que celle-ci s’y trouve.

Effet Le rôdeur des toiles repère automatiquement la créature et Marche rapidement, Escalade ou Descend sur un fil de toile avant de faire son jet d’initiative.
