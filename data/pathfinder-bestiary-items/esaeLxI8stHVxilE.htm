Parent Name: Jungle Drake Name: Twisting Tail Nom: Queue sinueuse État: officielle ------ Description (en) ------

Trigger A creature within reach of the jungle drake's stinger uses a move action or leaves a square during a move action it's using.

Effect The jungle drake Strikes the target with its stinger. If it hits, the jungle drake disrupts the creature's action.

------ Description (fr) ------

Déclencheur Une créature à portée du dard du drake utilise une action de déplacement ou quitte une case pendant une action de déplacement qu’il utilise.

Effet Le drake des jungles Frappe la cible avec son dard. S’il touche, le drake interrompt l’action de cette créature.
