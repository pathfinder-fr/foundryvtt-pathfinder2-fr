Parent Name: Roper Name: Flurry of Strands Nom: Déluge de filaments État: officielle ------ Description (en) ------

The roper makes a strand Strike with each of its strands (except those that are @Compendium[pf2e.conditionitems.eIcWbB5o3pP6OIMe]{Immobilizing} creatures). Each attack must be against a different target. These attacks count toward the roper's multiple attack penalty, but its multiple attack penalty doesn't increase until after all the attacks.

------ Description (fr) ------

L’enlaceur porte une Frappe de filament avec chacun de ses filaments (sauf ceux qui servent à @Compendium[pf2e.conditionitems.eIcWbB5o3pP6OIMe]{Immobiliser} des créatures). Chaque attaque doit être dirigée contre une cible différente. Ces attaques sont toutes comptabilisées dans le malus d’attaques multiples de l’enlaceur, mais son malus n’augmente qu’une fois que toutes les attaques ont été effectuées.
