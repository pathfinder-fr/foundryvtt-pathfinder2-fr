Parent Name: Lich Name: Drain Soul Cage Nom: Canaliser le phylactère État: officielle ------ Description (en) ------

Frequency once per day

Effect The lich taps into its @Compendium[pf2e.equipment-srd.NEbr7nKuiluJgBHT]{Soul Cage's} power to cast any arcane spell up to 6th level, even if the spell being cast is not one of the lich's prepared spells. The lich's soul cage doesn't need to be present for the lich to use this ability.

------ Description (fr) ------
