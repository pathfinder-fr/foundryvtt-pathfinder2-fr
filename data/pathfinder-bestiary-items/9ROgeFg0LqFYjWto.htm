Parent Name: Arboreal Regent Name: Awaken Tree Nom: État: aucune ------ Description (en) ------

The arboreal regent causes a tree within 180 feet to uproot itself and fight as a minion using the statistics for an @Compendium[pf2e.pathfinder-bestiary.oBMIc2S8ekmDgPpi]{Awakened Tree}. The arboreal regent can control up to two awakened trees at a time, and it can issue commands to both trees as a single action, which has the concentrate and auditory traits.
