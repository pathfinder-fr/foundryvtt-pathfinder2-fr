Parent Name: Veiled Master Name: Delayed Suggestion Nom: Suggestion retardée État: officielle ------ Description (en) ------

When a veiled master successfully casts @Compendium[pf2e.spells-srd.OsOhx3TGIZ7AhD0P]{Dominate} on a creature, a @Compendium[pf2e.spells-srd.qwlh6aDgi86U3Q7H]{Suggestion} spell triggers when the dominate spell ends. This suggestion usually causes the target to return to the veiled master, so the creature can cast dominate again, but a veiled master can set the suggestion to different orders if it wishes.

------ Description (fr) ------

Quand un maître voilé réussit à lancer le sort @Compendium[pf2e.spells-srd.OsOhx3TGIZ7AhD0P]{Domination} sur une créature, le sort @Compendium[pf2e.spells-srd.qwlh6aDgi86U3Q7H]{Suggestion} se déclenche à la fin de la durée du sort Domination. Généralement, cette Suggestion pousse la cible à revenir auprès du maître voilé pour que ce dernier puisse de nouveau lancer domination ; cependant, s’il le souhaite, le maître voilé peut définir d’autres ordres pour cette Suggestion.
