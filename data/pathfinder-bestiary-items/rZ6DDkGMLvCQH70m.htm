Parent Name: Giant Flytrap Name: Hungry Flurry Nom: Appétit frénétique État: officielle ------ Description (en) ------

The flytrap makes four leaf Strikes at a -2 penalty, each against a different target. These attacks count toward the flytrap's multiple attack penalty, but the multiple attack penalty doesn't increase until after it makes all its attacks.

------ Description (fr) ------

L’attrape-mouche effectue deux Frappes de feuille avec une pénalité de -2, chaque attaque étant dirigée contre une cible différente. Ces attaques comptent pour la pénalité d’attaques multiples de l’attrape-mouche mais ce malus n’augmente qu’une fois toutes les attaques effectuées.
