Parent Name: Erinys Name: Furious Fusillade Nom: Déferlement furieux État: officielle ------ Description (en) ------

The erinys hovers in place if they are flying and fires one arrow at any number of creatures in a @Template[type:cone|distance:30]. Each attack is rolled separately. This counts as one attack for the purpose of the erinys's multiple attack penalty.

------ Description (fr) ------

L’érinye reste en vol stationnaire et tire une flèche sur toutes les créatures situées dans un @Template[type:cone|distance:30]{cône de 9 m}. Chaque attaque est effectuée séparément. Cela compte comme une attaque pour le malus d’attaques multiples de l’érinye.
