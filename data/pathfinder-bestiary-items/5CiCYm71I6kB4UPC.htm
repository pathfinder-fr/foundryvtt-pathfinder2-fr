Parent Name: Pit Fiend Name: Masterful Quickened Casting Nom: Incantation accélérée magistrale État: officielle ------ Description (en) ------

Frequency once per round.

Effect If the pit fiend's next action is to cast an 8th-level or lower innate spell, reduce the number of actions to cast it by 1 (minimum 1 action).

------ Description (fr) ------

Fréquence une fois par round.

Effet Si à sa prochaine action le diantrefosse lance un sort inné de niveau 8 ou inférieur, réduisez de 1 le nombre d’actions pour le lancer (minimum 1 action).
