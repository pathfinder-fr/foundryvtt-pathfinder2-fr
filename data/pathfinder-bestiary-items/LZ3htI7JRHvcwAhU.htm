Parent Name: Baomal Name: Double Opportunity Nom: Double opportunité État: officielle ------ Description (en) ------

A baomal gains an extra reaction each round that it can use only to make an Attack of Opportunity. It must use a different head for each one it attempts, and it can't make more than one Attack of Opportunity for the same triggering action.

------ Description (fr) ------

Un baomal gagne une réaction supplémentaire au cours de chaque round pendant lequel il peut utiliser une Attaque d’opportunité. Il doit utiliser une tête différente pour chaque réaction qu’il tente et il ne peut effectuer plus d’une Attaque d’opportunité à cause du même déclencheur.
