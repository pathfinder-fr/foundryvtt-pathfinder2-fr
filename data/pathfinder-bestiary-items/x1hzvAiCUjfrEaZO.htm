Parent Name: Goliath Spider Name: Descend on a Web Nom: Descendre sur un fil de toile État: officielle ------ Description (en) ------

The goliath spider moves straight down up to 120 feet, suspended by a web line. It can hang from the web or drop off. The distance it Descends on a Web doesn't count for falling damage.

A creature that successfully Strikes the web (AC 20, Hardness 5, 20 HP) severs it, causing the spider to fall.

------ Description (fr) ------

L’araignée goliath descend à la verticale d’une distance maximale de 36 m, suspendue à un fil de toile. Elle peut rester accrochée au fil ou s’en détacher. La distance descendue avec cette action ne compte pas dans le calcul des dégâts de chute.

Une créature qui réussit à Frapper le fil de toile (CA 20, Solidité 5, 20 PV) le tranche et fait chuter l’araignée.
