Parent Name: Shadow Name: Light Vulnerability Nom: Vulnérabilité à la lumière État: officielle ------ Description (en) ------

An object shedding magical light (such as from the @Compendium[pf2e.spells-srd.WBmvzNDfpwka3qT4]{Light} spell) is treated as magical when used to attack the shadow.

------ Description (fr) ------

Un objet qui projette une lumière magique (comme celle créée par le sort @Compendium[pf2e.spells-srd.WBmvzNDfpwka3qT4]{Lumière}) est considéré comme magique s’il est utilisé pour attaquer une ombre.
