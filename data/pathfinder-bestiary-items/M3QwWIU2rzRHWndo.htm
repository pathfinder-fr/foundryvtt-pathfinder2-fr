Parent Name: Mimic Name: Mimic Object Nom: Imitation d'objet État: officielle ------ Description (en) ------

The mimic assumes the shape of any Medium object. This doesn't change the mimic's texture or overall size but can alter its coloration and visual appearance. It has an automatic result of 28 on Deception checks and DCs to pass as the object that it's mimicking.

------ Description (fr) ------

La mimique prend la forme d’un objet de taille Moyenne. Cela ne change pas sa texture ni sa taille globale, mais cela modifie sa coloration et son apparence visuelle. Elle obtient automatiquement 28 aux tests de Duperie et aux DD pour se faire passer pour l’objet qu’elle imite.
