Parent Name: Hydra Name: Storm of Jaws Nom: Tempêtes de machoires État: officielle ------ Description (en) ------

The hydra makes a number of Strikes up to its number of heads, each against a different target. These attacks count toward the hydra's multiple attack penalty, but the multiple attack penalty doesn't increase until after the hydra makes all its attacks.

------ Description (fr) ------

L’hydre effectue un nombre de Frappes au maximum égal à son nombre de têtes, chacune contre une cible différente. Ces attaques comptent pour le malus d’attaques multiples de l’hydre mais ce malus n’augmente qu’une fois toutes les attaques effectuées.
