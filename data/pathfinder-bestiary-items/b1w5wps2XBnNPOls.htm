Parent Name: Gug Name: Furious Claws Nom: Furie de griffes État: officielle ------ Description (en) ------

The gug makes up to four claw Strikes, each against a different target. These attacks all count toward the gug's multiple attack penalty, but the penalty doesn't increase until after the gug makes all its attacks.

------ Description (fr) ------

Le gug effectue jusqu’à quatre Frappes de griffe, chaque attaque sur une cible différente. Ces attaques comptent pour le malus d’attaques multiples du gug mais ce malus n’augmente qu’une fois toutes les attaques effectuées.
