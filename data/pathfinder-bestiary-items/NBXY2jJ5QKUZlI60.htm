Parent Name: Crimson Worm Name: Rock Tunneler Nom: Foreur de roche État: officielle ------ Description (en) ------

A crimson worm can burrow through solid stone at a Speed of 20 feet. It can leave a tunnel if it desires, and it usually does.

------ Description (fr) ------

Un ver écarlate peut forer la roche solide à une Vitesse de 6 m. Il peut laisser un tunnel derrière lui s’il le souhaite, ce qui est généralement le cas.
