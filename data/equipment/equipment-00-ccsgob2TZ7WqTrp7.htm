Name: Ten-Foot Pole Nom: Perche de 3 mètres État: officielle ------ Description (en) ------

When wielding this long pole, you can use Seek to search a square up to 10 feet away. The pole is not sturdy enough to use as a weapon.

------ Description (fr) ------

Lorsque vous utilisez cette perche, vous pouvez utiliser Chercher pour fouiller une case jusqu’à 3 mètres. La perche n’est cependant pas assez solide pour qu’on puisse l’utiliser comme une arme.
