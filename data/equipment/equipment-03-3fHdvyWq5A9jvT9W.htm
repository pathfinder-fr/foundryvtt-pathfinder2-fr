Name: Concealed Holster Nom: Étui d'arme dissimulé État: changé État d'origine: libre ------ Description (en) ------ NE PAS TRADUIRE ------ Description (fr) ------

Cet étui en cuir est conçu pour mieux dissimuler les petites armes à feu à la vue. Seules les armes à feu conçues pour être utilisées dans une seule main sont suffisamment petites pour être cachées avec cet étui. Vous gagnez un bonus d'objet de +1 aux tests de Discrétion et aux DD pour cacher ou dissimuler une arme à feu ou une arbalète à main dans cet étui.
