Name: Exploration Lens (Investigating) Nom: Lentille d'exploration d'enquête État: libre ------ Description (en) ------

Activate:1 Interact

An exploration lens is a magical glass lens designed for a @Compendium[pf2e.equipment-srd.QrNvP9SgnK9DrerA]{Bull's Eye Lantern}. When light from the lantern passes through the lens, it alters the light to make it easier to perform a common exploration activity with the lantern.

Alternatively, an exploration lens can be installed in a @Compendium[pf2e.equipment-srd.gbwr57aT9ou8yKWT]{Wayfinder} like an aeon stone. In this case, light (such as that from a @Compendium[pf2e.equipment-srd.dIRZ0LL7G31fJNYz]{Lantern (Hooded)}, @Compendium[pf2e.equipment-srd.8Jdw4yAzWYylGePS]{Torch}, or @Compendium[pf2e.spells-srd.WBmvzNDfpwka3qT4]{Light} spell) must still pass through the lens and you must hold the wayfinder in your hand for it to function.

There are several different types of exploration lens, each granting benefits with a different type of exploration activity. Each type of lens is distinguishable from all the other types by holding it up to a bright light.

Type: Investigating

The outer rim of this lens glows faintly when held up to the light.

While activated, the lens grants you a +1 item bonus to skill checks to Recall Knowledge while @Compendium[pf2e.actionspf2e.EwgTZBWsc8qKaViP]{Investigating} by outlining potential clues in a way that triggers your memory.

If you're Investigating something while using the lens, you can always roll a Recall Knowledge skill check appropriate to the investigation for initiative.

------ Description (fr) ------

Activation1 Interagir

Une lentille exploratrice est une lentille de verre magique conçue pour une @Compendium[pf2e.equipment-srd.QrNvP9SgnK9DrerA]{Lanterne sourde}. Lorsque la lumière de la lanterne passe à travers la lentille, elle modifie la lumière pour faciliter l'exécution d'une activité d'exploration courante avec la lanterne.

De manière alternative, une lentille exploratrice peut être installée dans un @Compendium[pf2e.equipment-srd.gbwr57aT9ou8yKWT]{Guide} comme une pierre d'éternité. Dans ce cas, la lumière (comme celle d'une @Compendium[pf2e.equipment-srd.dIRZ0LL7G31fJNYz]{Lanterne à capote}, d'une @Compendium[pf2e.equipment-srd.8 Jdw4yAzWYylGePS]{Torche} ou d'un sort de @Compendium[pf2e.spells-srd.WBmvzNDfpwka3qT4]{Lumière}) doit néanmoins toujours passer à travers la lentille et vous devez tenir le Guide dans votre main pour qu'il fonctionne.

Il existe plusieurs types de lentilles exploratrice, chacune accordant des avantages avec un type d'activité d'exploration différent. Chaque type de lentille se distingue de tous les autres types en la tenant devant une lumière vive.

Type : Enquête

Le bord extérieur de cette lentille brille faiblement lorsqu'on la tient à la lumière.

Lorsqu'elle est activée, la lentille vous accorde un bonus d'objet de +1 aux tests de compétence pour Se souvenir des connaissances lors d'une @Compendium[pf2e.actionspf2e.EwgTZBWsc8qKaViP]{Enquête} en soulignant les indices potentiels d'une manière qui déclenche votre mémoire.

Si vous enquêtez sur quelque chose tout en utilisant l'objectif, vous pouvez toujours lancer un test de compétence de Se souvenir approprié à l'enquête pour lancer l'initiative.
