Name: Storyteller's Opus Nom: Opus du bonimenteur État: changé État d'origine: libre ------ Description (en) ------ NE PAS TRADUIRE ------ Description (fr) ------

Ce grimoire vert avec une bordure dorée contient une histoire édifiante concernant un garçon qui a été dévoré par un loup après avoir précédemment menti sur les attaques de loups. Lorsqu'on l'ouvre, les gravures sur le devant semblent se déplacer, mettant en scène cette histoire en boucle.

Activation r Imaginer/p>

Fréquence une fois par jour

Déclencheur Une créature réussit, mais pas de manière critique, un test de Perception pour percer un sort d'illusion que vous avez préparé à partir de ce grimoire

Effet Vous racontez rapidement un bobard pour tenter de faire disparaître les incohérences de votre illusion. Tentez un test de Duperie contre le DD de Perception de la créature déclencheuse. Si vous réussissez, la créature ne perce pas l'illusion.
