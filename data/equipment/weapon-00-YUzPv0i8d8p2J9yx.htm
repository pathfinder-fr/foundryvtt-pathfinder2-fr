Name: Bola Nom: Bolas État: changé État d'origine: libre ------ Description (en) ------

This throwing weapon consists of weights tied to the end of long cords, which can be used to bludgeon foes or entangle their legs.

------ Description (fr) ------

Cette arme de jet est faite de poids reliés au bout de longues cordes, qui peuvent être utilisés comme armes contondantes contre des ennemis ou pour se prendre dans leurs jambes.
