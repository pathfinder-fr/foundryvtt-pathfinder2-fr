Activation 1 Interagir

Une lentille exploratrice est une lentille de verre magique conçue pour une @Compendium[pf2e.equipment-srd.QrNvP9SgnK9DrerA]{Lanterne sourde}. Lorsque la lumière de la lanterne passe à travers la lentille, elle modifie la lumière pour faciliter l'exécution d'une activité d'exploration courante avec la lanterne.

Alternativement, une lentille exploratrice peut être installée dans un @Compendium[pf2e.equipment-srd.gbwr57aT9ou8yKWT]{Guide} comme une pierre d'éternité. Dans ce cas, la lumière (comme celle d'une @Compendium[pf2e.equipment-srd.dIRZ0LL7G31fJNYz]{Lanterne à capote}, d'une @Compendium[pf2e.equipment-srd.8 Jdw4yAzWYylGePS]{Torche} ou d'un sort de @Compendium[pf2e.spells-srd.WBmvzNDfpwka3qT4]{Lumière}) doit toujours passer à travers la lentille et vous devez tenir le guide dans votre main pour qu'il fonctionne.

Il existe plusieurs types de lentilles exploratrice, chacune accordant des avantages avec un type d'activité d'exploration différent. Chaque type de lentille se distingue de tous les autres types en la tenant devant une lumière vive.

Type Chercher

Cette lentille a un éclat opalescent lorsqu'elle est tenue à la lumière.

La lentille souligne subtilement les portes cachées et les dangers dissimulés, vous accordant un bonus d'objet de +2 aux tests de Perception pour repérer de tels éléments au cours de l'activité @Compendium[pf2e.actionspf2e.TiNDYUGlMmxzxBYU]{Chercher}.

Même si vous ne Cherchez pas, vous bénéficiez d'un test pour trouver les pièges qui nécessitent normalement que vous les cherchiez.
