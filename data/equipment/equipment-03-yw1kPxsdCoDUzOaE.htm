Name: Musical Instrument (Virtuoso heavy) Nom: Instrument de musique (Virtuose lourd) État: officielle ------ Description (en) ------

Handheld instruments include bagpipes, a small set of chimes, small drums, fiddles and viols, flutes and recorders, small harps, lutes, trumpets, and similarly sized instruments. The GM might rule that an especially large handheld instrument (like a tuba) has greater Bulk. Heavy instruments such as large drums, a full set of chimes, and keyboard instruments are less portable and generally need to be stationary while played. A virtuoso instrument gives a +1 item bonus to Performance checks using that instrument.

Automatically applied effects:

  • +1 item bonus to Performance checks using that instrument.
------ Description (fr) ------ Les instruments à main comprennent les cornemuses, les petits carillons, les petits tambours, les violons et les violes, les flûtes et les flûtes à bec, les petites harpes, les luths, les trompettes et les instruments de taille similaire. Le MJ peut décider qu'un instrument à main particulièrement grand (comme un tuba) a plus d'impact. Les instruments lourds tels que les gros tambours, un ensemble complet de carillons et les instruments à clavier sont moins portables et doivent généralement rester immobiles lorsqu'ils sont joués. Un instrument virtuose confère un bonus d'élément de +1 aux tests de performance utilisant cet instrument.

Effets appliqués automatiquement:

  • Bonus d'objet de +1 aux tests de Représentation utilisant cet instrument.
