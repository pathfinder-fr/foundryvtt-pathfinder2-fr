Name: Warcaller's Chime of Restoration Nom: Carillon de rétablissement du belliqueux État: libre ------ Description (en) ------

This chime's carvings depict blooming flowers. When activated, it creates a small aura of healing in a @Template[type:emanation|distance:5]. Creatures in the aura gain fast healing 15. This aura remains active for 3 rounds. This chime can be used once per week.

------ Description (fr) ------

Les sculptures de ce carillon représentent des fleurs épanouies. Lorsqu'il est activé, il crée une petite aura de soins dans une @Template[type:emanation|distance:5]{émanation de 1,5 mètres}. Les créatures situées dans l'aura gagnent guérison accélérée 15. Cette aura reste active pendant 3 rounds. Ce carillon peut être utilisé une fois par semaine.
