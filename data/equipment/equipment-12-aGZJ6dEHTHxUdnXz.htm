Name: Warcaller's Chime of Restoration Nom: Carillon de rétablissement du belliqueux État: changé État d'origine: libre ------ Description (en) ------ NE PAS TRADUIRE ------ Description (fr) ------

Les sculptures de ce carillon représentent des fleurs épanouies. Lorsqu'il est activé, il crée une petite aura de soins dans une émanation de 1,5 mètres. Les créatures situées dans l'aura gagnent guérison accélérée 15. Cette aura reste active pendant 3 rounds. Ce carillon peut être utilisé une fois par semaine.
