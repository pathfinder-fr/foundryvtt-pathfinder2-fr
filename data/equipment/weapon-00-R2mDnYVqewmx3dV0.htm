Name: Exquisite Sword Cane Sheath Nom: Fourreau de canne épée raffinée État: libre ------ Description (en) ------

An exquisite sword cane is a sword sheathed inside a hollow cane, which itself can be used as a clubbing weapon with or without the sword sheathed inside.

------ Description (fr) ------

Une canne épée raffinée est une épée dont la lame se range à l'intérieur d'une canne évidée, ce qui permet à cette dernière d'être utilisée comme une trique avec ou sans l'épée rangée à l'intérieur.
