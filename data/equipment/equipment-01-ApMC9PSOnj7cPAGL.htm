Name: Bomb Launcher Nom: Lanceur de bombe État: changé État d'origine: libre ------ Description (en) ------ NE PAS TRADUIRE ------ Description (fr) ------

Ce long tube creux est tenu à deux mains, et placé sur l'épaule. À l'intérieur, il contient un petit panier métallique conçu pour contenir une bombe alchimique. Une glissière située sur le dessus du tube permet de déposer une bombe alchimique dans le panier interne, tandis qu'un levier situé placé sous le tube permet de tirer le panier vers l'arrière pour le verrouiller en position. Le chargement d'une bombe alchimique dans un lanceur de bombes nécessite une unique action d'Interaction. En appuyant sur la gâchette, le panier est précipité vers l'avant, ce qui vous permet de Frapper avec la bombe alchimique chargée sur de longues distances. Les bombes tirées avec un lanceur de bombes ont un facteur de portée de 18 mètres.
