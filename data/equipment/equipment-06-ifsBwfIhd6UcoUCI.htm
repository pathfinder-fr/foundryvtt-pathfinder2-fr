Name: Bellflower Toolbelt Nom: Ceinture à outil Campanule État: changé État d'origine: libre ------ Description (en) ------

Different versions of this belt are customized to appear to suit specific trades, so a belt used for carpentry would look somewhat different than a belt for baking.

Activate D envision, Interact

Effect You place an object of up to 1 Bulk into the belt, transforming that object into a tool befitting the trade for which the belt was created. As long as there are enough transformed items, the belt can be used as artisan's tools for that trade.

Each object remains transformed until it has been removed from the belt for 24 hours or someone uses a single Interact action to return it to its normal form.

------ Description (fr) ------

Différentes versions de cette ceinture sont adaptées pour correspondre à divers métiers de manière à ce qu'une ceinture utilisée par un charpentier se distingue de celle d'un cuisinier.

Activation imaginer, Intéragir; Effect Vous placez un objet jusqu'à 1 Encombrement dans la ceinture, le transformant en un outil correspondant au métier pourr lequel la ceinture a été créée. Tant que vous disposez d'assez de'outils transformés, la ceinture peut être utilisée comme des outils d'artisan pour exercer ce métier.

Chaque objet reste transformé jusqu'à ce qu'il soit ôté de la ceinture pendant 24 heures ou que quelqu'un utilise une unique action Intéragir pour le faire revenir à sa forme normale.
