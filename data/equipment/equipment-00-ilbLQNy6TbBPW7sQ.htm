Name: Clothing (Ordinary) Nom: Vêtements (ordinaire) État: officielle ------ Description (en) ------

Ordinary clothing is functional with basic tailoring, such as peasant garb, monk's robes, or work clothes.

------ Description (fr) ------

Les vêtements ordinaires sont fonctionnels avec une confection de base, comme les vêtements de paysan, les robes de moine ou les vêtements de travail.
