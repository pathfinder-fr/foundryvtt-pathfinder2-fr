Name: Hand Cannon Nom: Couleuvrine État: libre ------ Description (en) ------

Popular among privateers and mercenaries in Goka, hand cannons are little more than a hardened tube with a handle and external ignition attached. A hand cannon can be used to fire almost anything that can be packed into its barrel. The wielder of a hand cannon can change the damage type granted by its modular trait as part of the same Interact action used to reload.

------ Description (fr) ------

Populaires parmi les corsaires et les mercenaires de Goka, les couleuvrines ne sont guère plus qu'un tube durci auquel sont attachés une poignée et un allumage externe. La couleuvrine peut être utilisée pour tirer presque tout ce qui peut être placé dans son canon. Le porteur d'une couleuvrine peut changer le type de dégât accordé par son trait modulaire dans le cadre de l'action Interagir utilisée pour recharger.
