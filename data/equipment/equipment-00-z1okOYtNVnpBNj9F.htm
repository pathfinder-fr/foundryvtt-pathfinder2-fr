Name: Religious Text Nom: Texte religieux État: officielle ------ Description (en) ------

This manuscript contains scripture of a particular religion. Some divine spellcasters, such as clerics, can use a religious text as a divine focus to use certain abilities and cast some spells. A religious text must be held in one hand to use it.

------ Description (fr) ------

Ce manuscrit contient les écritures d’une religion spécifique. Certains incantateurs divins, tels que les prêtres, peuvent utiliser un texte religieux comme focaliseur divin pour utiliser certaines capacités et lancer des sorts. Un texte religieux doit être tenu dans une main pour pouvoir être utilisé.
