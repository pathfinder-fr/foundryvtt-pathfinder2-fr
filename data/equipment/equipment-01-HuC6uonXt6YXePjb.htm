Name: Armor Latches Nom: Loquets d'armure État: libre ------ Description (en) ------

This armor is easily doffed. A set of armor with armor latches gains the noisy trait. You cannot add armor latches to armor that already possesses the noisy trait. You can remove a set of armor with armor latches with a three-action activity, which has the manipulate trait. This doesn't affect the time it takes to don the armor.

------ Description (fr) ------

Cette armure peut être facilement enlevée. Un ensemble d'armure avec des loquets d'armure gagne le trait bruyant. Vous ne pouvez pas ajouter de loquets d'armure à une armure qui possède déjà le trait bruyant. Vous pouvez retirer l'ensemble de l'armure possédant des loquets d'armure avec une activité à trois actions, qui possède le trait manipulation. Cela n'affecte pas le temps nécessaire pour enfiler l'armure.
