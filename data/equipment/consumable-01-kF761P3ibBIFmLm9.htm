Name: Caltrop Snare Nom: Piège artisanal de chausse-trappes État: changé État d'origine: officielle ------ Description (en) ------ NE PAS TRADUIRE ------ Description (fr) ------

Ce piège artisanal se compose d’un récipient caché rempli de chausse-trappes et relié à un fil de détente. Quand il se déclenche, le récipient déverse les chausse-trappes dans la case où se trouve le piège ou dans une case adjacente. Vous désignez la case dans laquelle vous posez le piège.

Si les chausse-trappes se déversent dans une case occupée par une créature, celle-ci doit immédiatement faire un test d’Acrobaties.

Conditions de FabricationFournir une boîte de chausse-trappes.
