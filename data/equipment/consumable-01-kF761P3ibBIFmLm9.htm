Name: Caltrop Snare Nom: Piège artisanal de chausse-trappes État: officielle ------ Description (en) ------

This snare consists of a hidden canister of caltrops attached to a trip wire. When the snare is triggered, it flings the caltrops into either the snare's square or a square adjacent to the snare. You choose which square when you set up the snare.

If the caltrops scatter into the same square as a creature, that creature must attempt the Acrobatics check immediately.

Craft Requirements Supply a container of caltrops.

------ Description (fr) ------

Ce piège artisanal se compose d’un récipient caché rempli de chausse-trappes et relié à un fil de détente. Quand il se déclenche, le récipient déverse les chausse-trappes dans la case où se trouve le piège ou dans une case adjacente. Vous désignez la case dans laquelle vous posez le piège.

Si les chausse-trappes se déversent dans une case occupée par une créature, celle-ci doit immédiatement faire le test d’Acrobaties.

Conditions de Fabrication Fournir une boîte de chausse-trappes.
