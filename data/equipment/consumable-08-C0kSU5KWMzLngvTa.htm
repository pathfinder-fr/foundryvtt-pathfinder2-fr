Name: Oil of Object Animation Nom: Huile d'animation d'objet État: libre ------ Description (en) ------

Activate 2 Interact

This fine golden oil comes in a small blue canister. Carefully spreading the oil over an object turns it into an animated object of the same type. For example, sprinkling it on a statue makes an animated statue. If the animated object's level would be greater than 3, the oil struggles to animate it and ultimately fails.

This animated object has the minion trait, meaning it can use 2 actions when you Command it. You can issue a Command for the current turn as part of the activation. The object remains animated for 1 minute before falling inert. As usual, you can have a maximum of four minions under your control.

------ Description (fr) ------

Activation 2 Interagir

Cette huile fine et dorée est présentée dans un petit bidon bleu. En répandant soigneusement l'huile sur un objet, vous le transformez en un objet animé du même type. Par exemple, en l'aspergeant sur une statue, on obtient une statue animée. Si le niveau de l’objet animé est supérieur à 3, l’huile peine à l’animer et finit par échouer.

Cet objet animé possède le trait sbire, ce qui signifie qu'il peut utiliser 2 actions lorsque vous lui donnez un ordre. Vous pouvez donner un ordre pour le tour en cours dans le cadre de l'activation. L'objet reste animé pendant 1 minute avant de tomber inerte. Comme d'habitude, vous pouvez avoir un maximum de quatre sbires sous votre contrôle.
