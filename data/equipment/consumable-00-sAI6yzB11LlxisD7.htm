Name: Grappling Bolt Nom: Carreau grappin État: libre ------ Description (en) ------

This small grappling hook is designed to be tied to a rope and fired from a crossbow. When fired, it has half the normal range increment for the weapon.

------ Description (fr) ------

Ce petit grappin est conçu pour être attaché à une corde et tiré à partir d'une arbalète. Lorsqu'il est tiré, il a la moitié de l'augmentation de portée normale de l'arme.
