Name: Effect: Glittering Snare (Success) Nom: Effet : Piège scintillant (Succès) État: libre ------ Description (en) ------

The creature takes a -1 status penalty to Stealth checks for 1 minute or until it uses an Interact action to remove the glitter.

------ Description (fr) ------

La créature subit une pénalité de statut de -1 aux tests de Discrétion pendant 1 minute ou jusqu'à ce qu'elle utilise une action d'interaction pour retirer les paillettes.
