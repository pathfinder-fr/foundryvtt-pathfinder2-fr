Name: Effect: Improvised Weapon Nom: Effet : Arme improvisée État: libre ------ Description (en) ------

Automatically applied effects:

  • -2 penalty to attack rolls with any weapon with an improvised trait.
------ Description (fr) ------

Applique une pénalité de -2 aux jets d'attaque avec toute arme qui porte le trait "improvised".

Choisir dans l'inventaire ou dans le compendium l'arme que vous voulez émuler : par exemple un "bâton" si vous avez une branche, une "matraque" si vous prenez une bouteille, une dague si vous prenez une "stalactite", placez le dans l'inventaire.

dans l'onglet détails de ladite arme, éditez les traits pour ajouter tout en bas "improvised" (en anglais) qui n'est pas actuellement dans la liste (une demande d'intégration est en cours).

glisser déposer l'effet sur le jeton de votre PJ.

Une petite animation apparaîtra pour vous dire que l'effet est actif. Une pelle apparaîtra. Cela vous permettra d'activer ou non l'effet lors de la Frappe avec l'arme improvisée.

Pour désactiver l'effet, un clic droit sur l'icône suffit pelle.
