Name: (Ghost) Pyre's Memory Nom: (Fantôme) Mémoire de bûcher funéraire État: libre ------ Description (en) ------

The ghost causes great gouts of flame to erupt without warning.

The ghost picks three 5-foot squares within 30 feet. Creatures that start in or enter one of these squares take [[/r {1d6}[fire]]]{1d6 fire damage} per 2 levels the ghost has, subject to a @Check[type:reflex|dc:|basic:true] save.

The flames last for 1 minute and don't set other things alight unless the ghost wishes. If the ghost uses this ability again, it dismisses any existing flames.

------ Description (fr) ------

Le fantôme fait jaillir sans avertissement de grandes éruptions de flammes.

Le fantôme choisit une case de 1,5 mètres dans une zone de 9 mètres. Les créatures qui commencent leur tour ou qui entrent dans une de ces cases subissent [[/r {1d6}[fire]]]{1d6 dégâts de feu} par tranche de 2 niveau du fantôme, avec un jet de @Check[type:reflex|dc:|basic:true].

Les flammes durent une minute et n'enflamment pas d'autres objets, sauf si le fantôme le souhaite. Si le fantôme utilise à nouveau cette capacité, il annule les flammes existantes.
