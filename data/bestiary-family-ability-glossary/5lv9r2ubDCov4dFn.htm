Name: (Ghost) Pyre's Memory Nom: (Fantôme) Mémoire de bûcher funéraire État: changé État d'origine: libre ------ Description (en) ------ NE PAS TRADUIRE ------ Description (fr) ------

Le fantôme fait jaillir sans avertissement de grandes éruptions de flammes.

Le fantôme choisit une case de 1,5 mètres dans une zone de 9 mètres. Les créatures qui commencent leur tour ou qui entrent dans une de ces cases subissent [[/r {1d6}[fire]]]{1d6 dégâts de feu} par tranche de 2 niveau du fantôme, avec un jet de sauvegarde Réflexes basique.

Les flammes durent une minute et n'enflamment pas d'autres objets, sauf si le fantôme le souhaite. Si le fantôme utilise à nouveau cette capacité, il annule les flammes existantes.
