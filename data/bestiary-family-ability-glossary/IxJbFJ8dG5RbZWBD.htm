Name: (Graveknight) Devastating Blast Nom: (Chevalier sépulcre) Déflagration dévastatrice État: changé État d'origine: officielle ------ Description (en) ------ NE PAS TRADUIRE ------ Description (fr) ------

Le chevalier sépulcre projette un cône d’énergie de 9 mètres.

Les créatures prises dans la zone d’effet subissent 1d12 dégâts plus 1d12 dégâts supplémentaires tous les deux niveaux du chevalier ((Réflexes basique ).

Le chevalier sépulcre peut utiliser ce pouvoir une fois tous les [[/br 1d4 #rounds]]{1d4 rounds}.

Les dégâts d’énergie sont du même type que pour armes du désastre ; Déflagration dévastatrice gagne le trait associé à ce type d’énergie.
