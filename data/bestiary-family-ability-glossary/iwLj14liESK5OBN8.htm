Name: (Ghost) Malevolent Possession Nom: (Fantôme) Possession malveillante État: officielle ------ Description (en) ------

The ghost attempts to possess an adjacent corporeal creature. This has the same effect as the @Compendium[pf2e.spells-srd.wU6hNzK8Yfqdmc8m]{Possession} spell, except since the ghost doesn't have a physical body, it is unaffected by that restriction of the spell.

------ Description (fr) ------

Le fantôme tente de posséder une créature tangible adjacente. Ce pouvoir produit le même effet que le sort possession, mais comme le fantôme n’a pas de corps physique, les restrictions du sort ne l’affectent pas.
