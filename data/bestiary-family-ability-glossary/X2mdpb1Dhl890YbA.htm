Name: (Worm That Walks) Swarm Shape Nom: (Ver-qui-marche) Forme de nuée État: changé État d'origine: officielle ------ Description (en) ------ NE PAS TRADUIRE ------ Description (fr) ------

Le ver-qui-marche s’effondre en une nuée de vers informe. Il lâche tous les objets qu’il tient ou porte. Tant qu’il se trouve ainsi fragmenté, le ver-quimarche ne peut pas utiliser d’actions pour attaquer ni lancer de sorts, mais il peut traverser n’importe quelle zone dans laquelle ses vers peuvent se déplacer individuellement sans avoir besoin de Se faufiler. Il peut utiliser la même action pour reprendre sa forme normale.
