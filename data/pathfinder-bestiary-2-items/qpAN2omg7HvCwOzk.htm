Parent Name: Water Orm Name: Undetectable Nom: Indétectable État: officielle ------ Description (en) ------

A water orm automatically tries to counteract any detection, revelation, or scrying divination attempted against it, using its Stealth modifier for the counteract check.

------ Description (fr) ------

Un orm aquatique tente automatiquement de contrer toute tentative de détection, de révélation ou de scrutation par divination qui le prend pour cible. Il utilise son modificateur de Discrétion pour le test de contre.
