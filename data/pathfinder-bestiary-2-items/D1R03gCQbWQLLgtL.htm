Parent Name: Wood Golem Name: Splinter Volley Nom: Volée d'échardes État: officielle ------ Description (en) ------

The wood golem makes up to four splinter Strikes, each against a different target.

These attacks count toward the wood golem's multiple attack penalty, but the multiple attack penalty doesn't increase until after the wood golem makes all of its attacks.

------ Description (fr) ------

Le golem de bois effectue jusqu’à quatre Frappes d’écharde, chacune dirigée contre une cible différente. Ces attaques comptent pour le malus d’attaques multiples du golem, mais ce malus n’augmente qu’une fois toutes les attaques effectuées.
