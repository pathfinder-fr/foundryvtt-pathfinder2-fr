Parent Name: Scythe Tree Name: Dead Tree Nom: Arbre mort État: officielle ------ Description (en) ------

Until the scythe tree acts, it appears to be a dead tree. It has an automatic result of 35 on Deception checks and DCs to pass as a dead tree.

------ Description (fr) ------

Jusqu’à ce qu’il agisse, l’arbre faux semble être mort. Il obtient un résultat automatique de 35 sur ses tests et DD de Duperie pour se faire passer pour un arbre mort.
