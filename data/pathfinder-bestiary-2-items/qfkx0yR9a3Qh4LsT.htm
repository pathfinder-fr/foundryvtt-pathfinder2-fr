Parent Name: Zelekhut Name: Double Attack Nom: Double attaque État: officielle ------ Description (en) ------

The zelekhut makes two chain Strikes, each targeting a different creature. Each Strike counts toward the zelekhut's multiple attack penalty, but the penalty doesn't increase until after it has made both attacks. If the zelekhut subsequently uses the @Compendium[pf2e.bestiary-ability-glossary-srd.BCLvAx4Pz4MLa2pu]{Knockdown} action, it affects all creatures it hit with Double Attack.

------ Description (fr) ------

Le zelekhut effectue deux Frappes de chaîne, chacune ciblant une créature différente. Chaque Frappe compte pour le malus d’attaques multiples du zelekhut mais ce malus n’augmente qu’une fois les deux attaques effectuées. Si le zelekhut enchaîne en utilisant l’action @Compendium[pf2e.bestiary-ability-glossary-srd.BCLvAx4Pz4MLa2pu]{Renversement}, il affecte toutes les créatures qu’il a touchées avec sa Double attaque.
