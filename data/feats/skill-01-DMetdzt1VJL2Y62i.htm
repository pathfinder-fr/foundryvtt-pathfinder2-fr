Name: Snare Crafting Nom: Fabrication de pièges artisanaux État: changé État d'origine: officielle ------ Description (en) ------

You can use the Craft activity to create snares, using the rules from page 244. When you select this feat, you add the formulas for four common snares to your formula book.

------ Description (fr) ------

Prérequis qualifié en Artisanat

Vous pouvez utiliser l’activité Artisanat pour créer des pièges, en utilisant les règles de la page 242. Quand vous sélectionnez ce don, vous ajoutez immédiatement les formules de quatre pièges ordinaires à votre formulaire.
