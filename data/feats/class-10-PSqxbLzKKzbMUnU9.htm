Name: Entwined Energy Ki Nom: Énergie ki entrelacée PrereqEN: PrereqFR: État: changé État d'origine: libre ------ Benefits ---- Benefits: Avantage: Vous ajoutez un type de dégâts d'énergie que vos sorts ki peuvent infliger ------ Description (en) ------ NE PAS TRADUIRE ------ Description (fr) ------

Vous pouvez transformer votre puissance intérieure en un type d'énergie pour renforcer vos capacités ki.

Lorsque vous obtenez ce don, vous pouvez choisir entre des dégâts d'acide, de froid, d'électricité, de feu ou soniques. Lorsque vous lancez @Compendium[pf2e.feats-srd.10DbphslCihq8mxQ]{Explosion ki}, vous pouvez choisir pour le sort qu'il inflige le type de dégâts que vous avez choisi au lieu de dégâts de force. Lorsque vous lancez @Compendium[pf2e.spells-srd.YDMOqndvYFu3OjA6]{Forme ki} ou @Compendium[pf2e.feats-srd.3a07jkWezSuORSpS]{Frappe ki}, ajoutez le type de dégâts que vous avez choisi à la liste des types de dégâts que vous pouvez sélectionner pour le sort.

Spécial À la discrétion du MJ, ce don peut modifier d'autres sorts ki.
