Name: Patch Job Nom: Travail de correction PrereqEN: |Vehicle Mechanic Dedication PrereqFR: Dévouement de Mécanicien de véhicule État: libre ------ Description (en) ------

You can patch up damage to an object with a quick fix. You gain the @Compendium[pf2e.feats-srd.xT593tHyPkumPuzz]{Improvised Repair} general feat. When you use Improvised Repair, you can attempt a Crafting check with the same DC as Repair. If you succeed, you can use the object as a normal object of its type until it takes damage rather than as a shoddy object of its type.

------ Description (fr) ------

Vous pouvez faire un travail de correction sur un objet avec une réparation rapide. Vous obtenez le don général @Compendium[pf2e.feats-srd.xT593tHyPkumPuzz]{Réparation improvisée}. Lorsque vous utilisez Réparation improvisée, vous pouvez tenter un test d'Artisanat avec le même DD que pour Réparer DC. En cas de succès, vous pouvez utiliser l'objet comme un objet normal de son type jusqu'à ce qu'il subisse de nouveau des dégâts plutôtt que de considérer qu'il s'agit d'un objet de son type de mauvaise qualité.
