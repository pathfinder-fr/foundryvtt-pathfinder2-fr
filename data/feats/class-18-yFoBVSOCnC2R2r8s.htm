Name: Eternal Composition Nom: Composition perpétuelle PrereqEN: maestro muse PrereqFR: muse virtuose État: officielle ------ Benefits and spoiler ---- Benefits: The world is a stage upon which you are always playing. Avantage: Vous avez un tour de magie de composition constamment actif. ------ Description (en) ------

The world is a stage upon which you are always playing. You are permanently quickened; you can use your extra action only to cast a composition cantrip that requires 1 action to cast. While in exploration mode, you can declare that you are performing an eligible composition cantrip while using any exploration tactic. Even before your first turn in a combat encounter, that cantrip is active as if you had cast it on your previous turn.

------ Description (fr) ------

Le monde est une scène sur laquelle vous n’arrêtez jamais de vous produire. Vous êtes accéléré de manière permanente ; vous pouvez utiliser votre action supplémentaire uniquement pour lancer un tour de magie de composition nécessitant 1 action pour le lancer. En mode exploration, vous pouvez déclarer que vous accomplissez un tour de magie de composition approprié tout en utilisant une tactique d’exploration. Même avant votre premier tour lors d’une rencontre de combat, ce tour de magie est actif comme si vous l’aviez lancé lors du tour précédent.
