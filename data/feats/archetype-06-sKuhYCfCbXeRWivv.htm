Vous êtes si habitué à manier un bouclier que vous pouvez le faire lorsque vous utilisez la main qui est en train de le tenir pour d'autres besoins.

La main que vous utilisez pour manier le bouclier compte comme une main libre pour les besoins de l'action @Compendium[pf2e.actionspf2e.pvQ5rY2zrtPI614F]{Interagir}. Vous pouvez aussi tenir un autre objet dans sa main (mais vous ne pouvez toujours pas l'utiliser pour manier une arme).

Cet avantage ne s'applique pas au @Compendium[pf2e.equipment-srd.ltundBNFAnP7bgPr]{pavois}, lequel est toujours trop encombrant.
