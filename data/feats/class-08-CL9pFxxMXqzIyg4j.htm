Avantage: Vous obtenez le sort ki Posture des vents violents. ------ Description (en) ------

You learn a mystical stance that lets you attack from a distance. You gain the @Compendium[pf2e.spells-srd.SkarN4VlNxSJSJNw]{Wild Winds Stance} ki spell. Increase the number of Focus Points in your focus pool by 1.

While entering the stance is a ki spell, the wind crash Strikes the stance grants are not, so you can use them as often as you like while in the stance.

------ Description (fr) ------

Vous apprenez une posture mystique qui vous permet d’attaquer à distance. Vous obtenez le sort ki @Compendium[pf2e.spells-srd.SkarN4VlNxSJSJNw]{Posture des vents violents}. Augmentez de 1 le nombre de points de focalisation de votre réserve.

Cette posture est un sort ki, ce qui n’est pas le cas des Frappes de coup de vent qu’elle confère ; vous pouvez donc les utiliser aussi souvent que vous le souhaitez tant que vous gardez cette posture.
