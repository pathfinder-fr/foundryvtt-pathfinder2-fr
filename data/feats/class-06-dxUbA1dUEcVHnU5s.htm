Name: Clockwork Celerity Nom: Célérité mécanique PrereqEN: |armor, construct, or weapon innovation PrereqFR: innovation d'arme, d'armure ou de créature artificielle État: changé État d'origine: libre ------ Description (en) ------ NE PAS TRADUIRE ------ Description (fr) ------

Déclencheur Votre tour commence.

Vous pouvez utiliser des dispositifs mécaniques instables dans votre innovation pour pousser votre invention à agir plus rapidement. Vous êtes @Compendium[pf2e.conditionitems.nlCjDvLMf2EkV2dl]{Accéléré} pour ce tour. La façon dont vous pouvez utiliser cette action supplémentaire dépend de votre innovation.

  • Arme Vous pouvez utiliser l'action supplémentaire pour Frapper avec votre innovation ou Recharger votre innovation.
  • Armure Vous pouvez utiliser l'action supplémentaire pour Faire un pas, Marcher rapidement ou utiliser une autre action de déplacement accordée par votre innovation (comme Nager si vous avez le don Armure de plongée).
  • Créature artificielle Vous pouvez utiliser l'action supplémentaire pour Diriger votre innovation créature artificielle (ou pour fournir 1 des actions si vous dépensez 2 actions pour Diriger votre créature artificielle).
