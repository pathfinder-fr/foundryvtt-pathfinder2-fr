Name: Standby Spell Nom: Sort en attente PrereqEN: |Spellstrike PrereqFR: Frappe de sort État: changé État d'origine: libre ------ Description (en) ------ NE PAS TRADUIRE ------ Description (fr) ------

Vous avez privilégié un sort pour l'utiliser dans votre technique de combat et vous l'avez suffisamment pratiqué pour ne plus avoir besoin de le préparer pour le lancer. Choisissez un sort dans votre grimoire que vous pourriez utiliser avec Frappe de sort et dont le niveau n'est pas supérieur à l'emplacement de sort de magus le plus élevé dont vous disposez. Vous pouvez lancer ce sort sans l'avoir préparé à l'avance en dépensant un emplacement de sort d'un niveau suffisant pour lancer le sort choisi. Vous pouvez modifier ce sort chaque fois que vous ajoutez de nouveaux sorts à votre grimoire ou en étudiant votre grimoire pendant 1 heure.
