Name: Order Spell Nom: Sort d'ordre État: changé État d'origine: officielle ------ Description (en) ------

Prerequisites Druid Dedication

You gain the initial order spell from your order. If you don't already have one, you gain a focus pool of 1 Focus Point, which you can Refocus by being one with nature. (For more on order spells, see page 131.).

------ Description (fr) ------

Prérequis Dévouement du druide

Vous gagnez le sort d’ordre initial de votre ordre. Si vous n’en avez pas déjà, vous gagnez une réserve de focalisation de 1 point de focalisation, que vous pouvez Refocaliser en ne faisant qu’un avec la nature (pour plus d’informations sur les sorts d’ordre, voir page 119).
