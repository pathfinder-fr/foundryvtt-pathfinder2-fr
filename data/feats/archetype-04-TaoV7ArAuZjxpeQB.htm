Name: Order Spell Nom: Sort d'ordre PrereqEN: Druid Dedication PrereqFR: Dévouement du druide État: officielle ------ Description (en) ------

You gain the initial order spell from your order. If you don't already have one, you gain a focus pool of 1 Focus Point, which you can Refocus by being one with nature.

------ Description (fr) ------

Vous gagnez le sort d’ordre initial de votre ordre. Si vous n’en avez pas déjà, vous gagnez une réserve de focalisation de 1 point de focalisation, que vous pouvez Refocaliser en vous harmonisant avec la nature.
