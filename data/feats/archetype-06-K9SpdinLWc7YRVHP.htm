Name: Captivating Intensity Nom: Intensité captivante PrereqEN: |Captivator Dedication PrereqFR: Dévouement de captivant État: changé État d'origine: libre ------ Description (en) ------ NE PAS TRADUIRE ------ Description (fr) ------

Votre capacité à captiver les autres a augmenté et vous pouvez utiliser vos capacités de captivaant plus souvent. Vous semblez ne jamais être à court lorsqu'il s'agit de fasciner ou de distraire. Vous pouvez lancer chaque sort inné occulte accordé par les dons de l'archétype de captivant une fois supplémentaire par jour. Vous pouvez le faire pour les sorts de tous les niveaux obtenus, autres que vos deux niveaux de sorts les plus élevés provenant de vos sorts de captivant sauf vos tours de magie. Si vous avez Captivation intensifiée, vous pouvez lancer le sort choisi une deuxième fois par jour, bien que l'une des deux fois où vous le lancez, il soit intensifié à deux niveaux de moins que votre plus haut niveau de sorts de captivant, tant que le sort peut être lancé à ce niveau de sort.
