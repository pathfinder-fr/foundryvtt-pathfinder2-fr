Avantage: Vos ailes peuvent battre pendant de brefs instants pour pouvoir vous élever dans l'air. ------ Description (en) ------ NE PAS TRADUIRE ------ Description (fr) ------

Vous manifestez des ailes qui peuvent battre pour de brefs instants. Vous n'avez besoin de dépenser aucune action supplémentaire pour atteindre quelque chose dans votre case qu'une créature de taille Moyenne pourrait atteindre. Par exemple, si vous voulez ouvrir une boîte de cookies située à 1,20 mètres au dessus du sol, vous avez seulement besoin d'une unique action d'Interaction pour y parvenir. Lorsque vous utilisez Ailes évanescentes pour vous élever vers un lieu plus haut dans votre case, votre action obtient le trait déplacement.
