Name: Terrain Expertise Nom: Expertise du terrain PrereqEN: trained in Survival PrereqFR: État: changé État d'origine: officielle ------ Description (en) ------

Your experience in navigating a certain type of terrain makes you supremely confident while doing so. You gain a +1 circumstance bonus to Survival checks in one of the following types of terrain, chosen when you select this feat: aquatic, arctic, desert, forest, mountain, plains, sky, swamp, or underground.

Special You can select this feat more than once, choosing a different type of terrain each time.

------ Description (fr) ------

Prérequis qualifié en Survie

Votre expérience de l’orientation sur un certain type de terrain vous rend extrêmement confiant pour ce faire. Vous gagnez un bonus de circonstances de +1 aux tests de Survie dans l’un des types de terrain suivants, choisi quand vous sélectionnez ce don : aquatique, arctique, désert, forêt, montagne, plaine, ciel, marais ou souterrain.

Spécial Vous pouvez sélectionner ce don plus d’une fois en choisissant un type de terrain différent à chaque fois.
