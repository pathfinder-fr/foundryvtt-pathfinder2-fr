Name: Moment of Clarity Nom: Moment de lucidité PrereqEN: PrereqFR: État: officielle ------ Benefits and spoiler ---- Benefits: You push back your rage for a moment in order to think clearly. Avantage: Vous repoussez votre colère un instant pour réfléchir clairement. ------ Description (en) ------

You push back your rage for a moment in order to think clearly. Until the end of this turn, you can use actions with the concentrate trait even if those actions don't have the rage trait.

------ Description (fr) ------

Vous contenez votre rage un moment afin de pouvoir penser avec lucidité. Jusqu’à la fin de ce tour, vous pouvez utiliser des actions avec le trait concentration, même si ces actions n’ont pas le trait rage.
