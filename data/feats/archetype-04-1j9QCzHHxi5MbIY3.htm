Name: Big Game Trapper Nom: Piégeur de gros gibier PrereqEN: |Snare Crafter|Game Hunter Dedication PrereqFR: Fabricant de pièges artisanaux|Dévouement de chasseur de gibier État: libre ------ Description (en) ------

You can set snares specifically for big prey, leaving smaller creatures unscathed. Each time you set a snare, you can choose Small, Medium, Large, or Huge. That snare can be triggered only by creatures of that size or larger.

------ Description (fr) ------

Vous pouvez poser des pièges spécialement conçus pour le gros gibier, en laissant indemnes les plus petites créatures. Chaque fois que vous posez un piège artisanal, vous pouvez choisir Petite, Moyenne, Grande ou Très grande. Ce piège artisanal ne peut être déclenché que par des créatures de cette taille ou de taille supérieure.
