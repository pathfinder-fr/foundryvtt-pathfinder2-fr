Name: Speak With Bats Nom: Parler avec les chauves-souris PrereqEN: nyktera heritage PrereqFR: État: changé État d'origine: libre ------ Description (en) ------

You can ask questions of, receive answers from, and use the Diplomacy skill with bats. You gain a +2 circumstance bonus on all Diplomacy checks to Make an Impression on bats or Request something from bats.

------ Description (fr) ------

Vous pouvez poser des questions, recevoir des réponses et utiliser la compétence Diplomatie avec les chauves-souris. Vous obtenez un bonus de circonstances de +2 à tous les tests de Diplomatie pour Faire bonne impression sur les chauves-souris ou Solliciter les chauves-souris.
