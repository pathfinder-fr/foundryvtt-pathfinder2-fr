When you use @Compendium[pf2e.feats-srd.Ewk7h9aQpKvy1RJo]{Crystalline Dust}, you can spend one additional action to disperse the motes as a @Template[type:emanation|distance:10]{10-foot-radius} cloud centered on you instead of as haze; the activity gains the concentrate trait.

------ Description (fr) ------

Lorsque vous utilisez @Compendium[pf2e.feats-srd.Ewk7h9aQpKvy1RJo]{Poussière cristalline}, vous pouvez dépenser une action supplémentaire pour disperser les particules dans un nuage de 3 m de rayon centré sur vous au lieu d'une brume ; l'activité obtient le trait concentration.
