Name: Improvised Critical Nom: Critique improvisé PrereqEN: Weapon Improviser Dedication PrereqFR: Dévouement d'improvisateur d'arme État: libre ------ Description (en) ------

You can apply critical specialization effects to improvised weapons you wield. For any given improvised weapon, the GM determines which critical specialization effect is most applicable, based on the type of object it is.

------ Description (fr) ------

Vous pouvez appliquer les effets spécialisés des coups critiques aux armes improvisées que vous maniez. Pour chaque arme improvisée donnée, le MJ détermine l'effet de critique spécialisé qui est le plus approprié en fonction du type d'objet.
