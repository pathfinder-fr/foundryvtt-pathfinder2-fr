Name: Shadow Hunter Nom: Chasseur de l'ombre État: officielle ------ Description (en) ------

Prerequisites Camouflage

You blend in to your surroundings so well that others have trouble telling you apart from the terrain. While in natural terrain, you’re always concealed from all foes if you choose to be, except for your hunted prey.

------ Description (fr) ------

Prérequis Camouflage

Vous vous fondez si parfaitement dans votre environnement qu’il est impossible de vous repérer. Dans un environnement naturel, vous êtes toujours masqué vis-à-vis de tous les adversaires si tel est votre souhait, sauf de votre proie.
