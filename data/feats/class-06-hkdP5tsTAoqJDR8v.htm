Name: Triple Shot Nom: Triple tir PrereqEN: Double Shot PrereqFR: Double tir État: changé État d'origine: officielle ------ Description (en) ------

You can quickly fire multiple shots with greater control.

When you use @Compendium[pf2e.feats-srd.PwcmmJOLY8C9JHau]{Double Shot}, you can make the attacks against the same target. You can add an additional action to Double Shot to make three ranged Strikes instead of two. If you do, the penalty is -4. All attacks count toward your multiple attack penalty, but the penalty doesn't increase until after you've made all of them.

------ Description (fr) ------

Vous avez amélioré la maîtrise de votre arme, ce qui vous permet d’effectuer plusieurs tirs très rapidement. Quand vous utilisez Double tir, vous pouvez effectuer les attaques contre la même cible.
