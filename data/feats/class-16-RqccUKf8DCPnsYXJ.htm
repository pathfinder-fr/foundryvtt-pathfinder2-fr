You can use @Compendium[pf2e.feats-srd.bh2jHyyYrkLMsIdX]{Prescient Planner} even if you have already used it after purchasing goods, and you can use it as a single action instead of a 2-action activity, during which you @Compendium[pf2e.actionspf2e.pvQ5rY2zrtPI614F]{Interact} to draw the item.
