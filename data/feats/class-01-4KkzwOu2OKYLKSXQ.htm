Name: Variable Core Nom: Noyau alternatif PrereqEN: PrereqFR: État: libre ------ Description (en) ------

You adjust your innovation's core, changing the way it explodes. When you choose this feat, select acid, cold, or electricity. Your innovation's core runs on that power source. When using the Explode action, or any time your innovation explodes on a critical failure and damages you, change the damage type from fire damage to the type you chose.

If you have the offensive boost class feature and retrain Variable Core to a different damage type, you can also switch to a different offensive boost that deals the same damage type if you wish.

------ Description (fr) ------

Vous ajustez le noyau de votre innovation, modifiant ainsi la façon dont elle explose. Lorsque vous choisissez ce don, sélectionnez l'acide, l'électricité ou le froid. Le noyau de votre innovation fonctionne avec cette source d'énergie. Lorsque vous utilisez l'action Exploser ou à chaque fois que votre innovation explose sur un échec critique et vous inflige des dégâts, changez le type de dégâts de feu pour le type que vous avez choisi.

Si vous possédez la capacité de classe Renfort offensif et que vous convertissez le Noyau alternatif pour un type de dégâts différent, vous pouvez également échanger pour un autre Renfort offensif qui inflige le même type de dégâts si vous le souhaitez.
