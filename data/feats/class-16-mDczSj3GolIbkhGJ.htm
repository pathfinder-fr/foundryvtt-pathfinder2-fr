Déclencheur Un ennemi vous rate avec une attaque à distance utilisant un type de munition approprié à votre arme (une balle si vous maniez une arme à feu).

Conditions Vous maniez une arme à feu ou une arbalète non chargées.

D'un seul geste fluide, vous attrapez un projectile en l'air, le chargez dans votre arme et le renvoyez sur l'attaquant. Interagissez pour charger le projectile dans votre arme, puis effectuez une Frappe avec l'arme requise contre l'adversaire déclencheur. Puisque vous utilisez les munitions de l'ennemi, cette Frappe applique tous les effets spéciaux que ces munitions pourraient avoir (par exemple, s'il s'agissait de @Compendium[pf2e.equipment-srd.rKknk8odXDBpON5l]{Munitions explosives}, elles exploseraient dans une explosion de feu si votre Frappe touche).
