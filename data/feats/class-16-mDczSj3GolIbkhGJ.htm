Name: Instant Return Nom: Retour instantané PrereqEN: PrereqFR: État: libre ------ Description (en) ------

Trigger An enemy misses you with a ranged attack using a type of ammunition appropriate to your weapon (a bullet if you're wielding a firearm).

Requirements You're wielding an unloaded firearm or crossbow.

With a single fluid gesture, you catch a projectile out of the air, load it in your weapon, and shoot it back at the attacker. Interact to load the projectile into your weapon, then make a Strike with the required weapon against the triggering opponent. Since you're using the foe's ammunition, this Strike applies any special effects that ammunition would have (for instance, if it was @Compendium[pf2e.equipment-srd.rKknk8odXDBpON5l]{Explosive Ammunition}, it would explode in a burst of fire if your Strike hits).

------ Description (fr) ------

Déclencheur Un ennemi vous rate avec une attaque à distance utilisant un type de munition approprié à votre arme (une balle si vous maniez une arme à feu).

Conditions Vous maniez une arme à feu ou une arbalète non chargées.

D'un seul geste fluide, vous attrapez un projectile en l'air, le chargez dans votre arme et le renvoyez sur l'attaquant. Interagissez pour charger le projectile dans votre arme, puis effectuez une Frappe avec l'arme requise contre l'adversaire déclencheur. Puisque vous utilisez les munitions de l'ennemi, cette Frappe applique tous les effets spéciaux que ces munitions pourraient avoir (par exemple, s'il s'agissait de @Compendium[pf2e.equipment-srd.rKknk8odXDBpON5l]{Munitions explosives}, elles exploseraient dans une explosion de feu si votre Frappe touche).
