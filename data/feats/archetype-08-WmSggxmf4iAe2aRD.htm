Name: Recycled Cogwheels Nom: Roues dentées recyclées PrereqEN: |Trapsmith Dedication PrereqFR: Dévouement de Fabricant de pièges État: libre ------ Description (en) ------

You're able to scavenge the cogwheels from your daily quick-deploy snares that use gears. This allows you to deconstruct a snare that didn't trigger in order to set the snare up somewhere else. Doing so takes the same number of actions as setting the snare did. When you do, you recover the snare and can deploy it in another location.

------ Description (fr) ------

Vous pouvez récupérer les roues dentées de vos pièges à déploiement rapide quotidiens qui utilisent des engrenages. Cela vous permet de déconstruire un piège qui ne s'est pas déclenché afin de l'installer ailleurs. Cela prend le même nombre d'actions que la pose du piège. Lorsque vous y parvenez, vous récupérez le piège et pouvez le déployer à un autre endroit.
