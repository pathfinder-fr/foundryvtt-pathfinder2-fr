Name: Incredible Luck (Halfling) Nom: Chance incroyable (Halfelin) PrereqEN: |Halfling Luck PrereqFR: Chance halfeline État: libre ------ Benefits ---- Benefits: You always seem to bounce back from the worst. Avantage: Vous pouvez utiliser Chance halfeline une fois par heure au lieu d'une fois par jour, mais uniquement sur vous. ------ Description (en) ------

Even other halflings consider you to be particularly lucky. You can use Halfling Luck once per hour, rather than once per day. If you have @Compendium[pf2e.feats-srd.k0sXAn4PPq5nW9al]{Guiding Luck}, you can still use Halfling Luck when you fail a Perception check or attack roll only once per day (though you can use it within the same hour that you used Halfling Luck), and if you have @Compendium[pf2e.feats-srd.f5Vkk2rM6tCe2zQn]{Shared Luck}, you can still use Halfling Luck on an ally instead of yourself only once per day.

------ Description (fr) ------

Même les autres halfelins vous considèrent particulièrement chanceux. Vous pouvez utiliser Chance halfeline une fois par heure au lieu d'une fois par jour.

Si vous avez le don @Compendium[pf2e.feats-srd.k0sXAn4PPq5nW9al]{Chance directrice}, vous pouvez toujours utiliser Chance halfeline lorsque vous manquez un test de Perception ou un jet d'attaque seulement une fois par jour (bien que vous puissiez l'utiliser durant la même heure au cours de laquelle vous avez utilisé Chance halfeline) et si vous avez le don @Compendium[pf2e.feats-srd.gWyCNTWUhxneOBne]{Halfelin utile}, vous pouvez toujours utiliser Chance halfeline sur un allié au lieu de vous-même une fois par jour seulement.
