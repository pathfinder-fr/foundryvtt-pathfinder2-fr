Name: Order Magic Nom: Magie des ordres État: officielle ------ Description (en) ------

Prerequisites Order Explorer

You have delved deeper into the teaching of a new order, gaining access to a coveted order spell. Choose an order you have selected with Order Explorer. You gain the initial order spell from that order.

Special You can take this feat multiple times. Each time you do, you must choose a different order you have selected with Order Explorer.

------ Description (fr) ------

Prérequis Explorateur des ordres

Vous vous êtes plongé dans l’étude approfondie des enseignements d’un nouvel ordre et avez découvert un sort d’ordre convoité. Choisissez un ordre que vous avez sélectionné avec Explorateur des ordres. Vous gagnez le sort initial de cet ordre.

Spécial Vous pouvez sélectionner ce don plusieurs fois. À chaque fois, vous devez choisir un ordre différent parmi ceux que vous avez choisis avec Explorateur des ordres.
