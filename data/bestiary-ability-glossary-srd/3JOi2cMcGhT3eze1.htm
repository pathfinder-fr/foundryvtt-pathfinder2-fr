Name: Rend Nom: Éventration État: changé État d'origine: officielle ------ Description (en) ------

A Rend entry lists a Strike the monster has.

Requirements The monster hit the same enemy with two consecutive Strikes of the listed type in the same round.

Effect The monster automatically deals that Strike's damage again to the enemy.

------ Description (fr) ------

L’entrée Éventration indique une Frappe que le monstre peut faire.

Conditions Au cours d’un round, le monstre réussit deux Frappes consécutives du type indiqué contre le même ennemi.

Effet Le monstre inflige une nouvelle fois automatiquement les dégâts de cette Frappe à cet ennemi.
