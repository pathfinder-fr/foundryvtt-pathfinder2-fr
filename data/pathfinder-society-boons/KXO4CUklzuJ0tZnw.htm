Name: Home Region Nom: État: changé État d'origine: auto-trad ------ Description (en) ------

[Slotless]

Prerequisites Tier 0 with any faction

Cost 0 Fame

Effect Whether it's because you grew up there or have since learned to call it home, one nation is especially familiar to you. Choose one nation (such as Varisia or Taldor) when you purchase this boon. For the purpose of fulfilling prerequisites and Access conditions for uncommon character options, you are treated as being from that nation as well as the larger region in which it's found (such as the Saga Lands for Varisia or the Shining Kingdoms for Taldor).

Special You can purchase this boon multiple times, but each time you purchase it, you lose your previous home region in order to become so familiar with a new one. Before doing so, you must retrain any options that listed being from the previous home region as a prerequisite, and any options to which you would no longer have access.

Achievement Points The World Traveler boon, accessible with Achievement Points, grants similar access to an entire region, providing an additional avenue to access region- and nation-based options.

------ Description (fr) ------

[Slotless]

Prérequis: Niveau 0 avec n'importe quelle faction

Coût : 0 Fame

Effet : Que ce soit parce que vous y avez grandi ou parce que vous avez appris à vous y sentir chez vous, une nation vous est particulièrement familière. Choisissez une nation (comme Varisia ou Taldor) lorsque vous achetez ce bienfait. Afin de remplir les conditions préalables et les conditions d'accès pour les options de personnage peu communes, vous êtes considéré comme étant de cette nation ainsi que de la plus grande région dans laquelle elle se trouve (comme les Terres de la Saga pour Varisia ou les Royaumes Lumineux pour Taldor).

Special : Vous pouvez acheter cette aubaine plusieurs fois, mais à chaque fois que vous l'achetez, vous perdez votre région d'origine précédente pour vous familiariser avec une nouvelle. Avant de le faire, vous devez recycler toutes les options pour lesquelles il est nécessaire d'être originaire de la région d'origine précédente, ainsi que toutes les options auxquelles vous n'aurez plus accès

Points d'accomplissement : Le boon Voyageur du monde, accessible avec des points d'accomplissement, accorde un accès similaire à une région entière, fournissant une voie supplémentaire pour accéder aux options basées sur la région et la nation.
