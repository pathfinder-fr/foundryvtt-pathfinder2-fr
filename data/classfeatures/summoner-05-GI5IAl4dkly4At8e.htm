Name: Ability Boosts Nom: Primes de caractéristiques (conjurateur) État: libre ------ Description (en) ------

At 5th level and every 5 levels thereafter, boost four different ability scores. You can use these boosts to increase ability scores above 18. Boosting an ability score increases it by 1 if it's already 18 or above, or by 2 if it starts out below 18.

Your eidolon also gets four ability boosts at these levels. The eidolon's ability boosts follow the same rules as yours.

------ Description (fr) ------

Au niveau 5, puis tous les cinq niveaux, vous améliorez quatre caractéristiques différentes. Vous pouvez utiliser ces primes de caractéristiques pour faire passer une valeur de caractéristique au-dessus de 18. La prime augmente la valeur de caractéristique de 1 si elle est déjà de 18 ou plus ou de 2 si elle est inférieure à 18.

Votre eidolon obtient aussi quatre primes de caractéristiques à ces niveaux. Les primes de caractéristiques suivent les mêmes règles que les vôtres.
