Name: Laughing Shadow Nom: Ombre ricanante État: changé État d'origine: libre ------ Description (en) ------ NE PAS TRADUIRE ------ Description (fr) ------

La magie est libératrice, un moyen de parvenir à vos fins, et vous pouvez l'utiliser pour aller où vous voulez, faire ce qui vous plaît et éviter les conséquences. Vous êtes une ombre ricanante de sort et de lame, toujours un pas devant vos adversaires, toujours avec un atout dans votre manche.

Lorsque vous adoptez la posture Cascade arcanique, vous obtenez un bonus de statut de +1,5 m à vos Vitesses ou un bonus de +3 m si vous ne portez pas d'armure. Si vous disposez d'une main libre lorsque vous êtes dans cette posture et que vous attaquez une créature @Compendium[pf2e.conditionitems.AJh5ex99aV6VTggg]{Prise au dépourvu}, vous augmentez les dégâts supplémentaires à 3, à 5 si vous disposez de spécialisation martiale ou à 7 si vous disposez de la spécialisation martiale supérieure. Vous devez avoir votre autre main complètement libre ; les dégâts supplémentaires ne s'appliquent pas si vous avez une arme permettant de disposer d'une main libre ou un autre objet dans cette main, même si vous devriez normalement être capable d'utiliser la main pour faire d'autres choses.
