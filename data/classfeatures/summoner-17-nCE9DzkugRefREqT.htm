Name: Eidolon Transcendence Nom: Transcendance de l'eidolon État: changé État d'origine: libre ------ Description (en) ------ NE PAS TRADUIRE ------ Description (fr) ------

Le lien forgé avec votre eidolon approche son paroxysme, accordant à votre eidolon la possibilité d'accéder à une nouvelle capacité puissante qui dépend de son type, comme indiqué dans les Capacités de l'eidolon pour chaque description de chaque type d'eidolon. Cette capacité accroit aussi l'un des scores de caractéristique de l'eidolon de 2.
