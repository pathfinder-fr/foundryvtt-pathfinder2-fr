Name: Swashbuckler's Style Nom: Style du bretteur État: changé État d'origine: libre ------ Description (en) ------

As a swashbuckler, you have your own distinctive style that enables you to gracefully handle everyday events and life-or-death situations alike. Choose a swashbuckler's style. This style determines the additional actions you use to gain panache and makes you trained in the skill related to that action.

------ Description (fr) ------

En tant que bretteur, vous possédez votre propre style distinctif qui vous permet de gérer avec élégance les événements quotidiens ainsi que les situations de vie ou de mort. Choisissez un style de bretteur. Ce style détermine les actions supplémentaires que vous pouvez utiliser pour gagner du panache et vous rend entraîné dans la compétence liée à cette action.
