Name: Greater Field Discovery (Chirurgeon) Nom: Découverte de domaine de recherche supérieure (Chirurgien) État: changé État d'origine: officielle ------ Description (en) ------

When you use Quick Alchemy to create any type of elixir of life, the creature drinking the elixir gains the maximum Hit Points possible for that elixir, instead of rolling to determine the number of Hit Points regained.

------ Description (fr) ------

Quand vous utilisez Alchimie rapide pour créer n’importe quel type d’élixir de vie, la créature qui le boit gagne le maximum de points de vie qu’il peut conférer au lieu de devoir lancer les dés pour déterminer combien de points de vie elle gagne.
