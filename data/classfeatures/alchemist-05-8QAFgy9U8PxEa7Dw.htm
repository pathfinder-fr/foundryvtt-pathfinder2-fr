Name: Field Discovery (Bomber) Nom: Découverte de domaine de recherche (Artificier) État: changé État d'origine: officielle ------ Description (en) ------

When using advanced alchemy to make bombs during your daily preparations, you can use a batch of reagents to create any three bombs instead of just two of the same bomb.

------ Description (fr) ------

Quand vous utilisez alchimie avancée pour faire des bombes lors de vos préparatifs quotidiens, vous pouvez utiliser un lot de réactifs pour créer trois bombes de n’importe quel type au lieu de deux bombes d’un seul type.
