Name: Unlimited Signature Spells Nom: Sorts emblématiques illimités État: changé État d'origine: libre ------ Description (en) ------ NE PAS TRADUIRE ------ Description (fr) ------

Tous vos sorts sont des sorts emblématiques. Cela signifie que si vous connaissez un sort, vous pouvez l'intensifier librement en le lançant à partir d'un emplacement de sort de niveau supérieur, jusqu'au maximum du niveau de sort que vous pouvez lancer. Vous pouvez de la même manière lancer tout sort dans une version inférieure sans avoir à les apprendre séparément.
