Name: Quick Alchemy Nom: Alchimie rapide État: officielle ------ Description (en) ------

If you need a specific alchemical item on the fly, you can use your infused reagents to quickly create it with the @Item[Quick Alchemy] action.

------ Description (fr) ------

Si vous avez besoin d’un objet alchimique à la volée, vous pouvez utiliser vos réactifs imprégnés pour le créer rapidement grâce à l’action Alchimie rapide.@Item[Alchimie rapide] action.
