Name: Luminous Sprite Nom: Sprite lumineux État: libre ------ Description (en) ------

You glow with magical luminance. You naturally shed light as a torch; this light has the evocation, light, and primal traits. You can extinguish, rekindle, or change the color of this light using a single action, which has the concentrate trait.

------ Description (fr) ------

Vous brillez avec une luminance magique. Vous diffusez naturellement de la lumière comme une torche ; cette lumière possède les traits évocation, lumière et primordial. Vous pouvez éteindre, rallumer ou changer la couleur de cette lumière en utilisant une unique action qui possède le trait concentration.
