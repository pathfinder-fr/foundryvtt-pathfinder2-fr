Parent Name: Hesperid Name: Sunset Dependent Nom: État: aucune ------ Description (en) ------

A hesperid is mystically bonded to a single remote location with a good view of the sunset—usually an island, coastal cliff, or valley. If they aren't at that location and able to see the sky at sunset on any given day, they become @Compendium[pf2e.conditionitems.4D2KBtexWXa6oUMR]{Drained 1}, increasing the value by 1 for each missed sunset and reducing by 1 only when they see the sunset.

A hesperid can perform a 24-hour ritual to bond to a new location.
