Name: Affix a Fulu Nom: Fixer un fulu État: changé État d'origine: libre ------ Description (en) ------ NE PAS TRADUIRE ------ Description (fr) ------

Vous fixez un fulu à une armure, une arme, un bouclier ou une créature ou une structure qui est à côté ou dans la même case que vous. Une créature peut retirer un fulu sur elle-même ou sur un objet abandonnéé dans cette portée par une action unique.
