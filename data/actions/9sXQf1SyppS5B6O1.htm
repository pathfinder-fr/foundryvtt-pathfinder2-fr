Vous stabilisez votre corps et observez les évènements sur le champ de bataille pour amximiser la portée de votre prochain tir. Vous réduisez la pénalité pour tirer dans le second facteur de portée de votre arme de -2 à 0 pour la prochaine attaque à distance que vous portez durant ce tour. Vous pouvez utiliser cette action une seconde fois durant le même tour pour réduire la pénalité pour tirer dans votre troisième facteur de portée de -4 à 0 pour la prochaine attaque à distance que vous portez durant ce tour.
